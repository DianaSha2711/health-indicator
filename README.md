Функция для определения уровня здоровья игрового персонажа.

## Описание

Функция принимает объект персонажа и возвращает его уровень здоровья:
- **healthy** (зелёный) - здоровье более 50
- **wounded** (жёлтый) - здоровье от 50 до 15 включительно
- **critical** (красный) - здоровье менее 15
